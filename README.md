# Tutor de Vocabulario

## Participantes
- Lucía González Mandler  
- Sofía Pérez Vásquez  

## Descripción
Programa en Python que permite practicar vocabulario mediante ejercicios de traducción. El usuario puede responder preguntas, añadir nuevas palabras y consultar su progreso.

## Funcionalidades
- Ejercicio de escritura (respuesta libre)
- Ejercicio tipo test (opción múltiple)
- Añadir tarjetas al mazo
- Ver estadísticas del usuario

## Ejecución
Ejecutar desde el archivo *main.py*
