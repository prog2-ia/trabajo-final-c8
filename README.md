## Descripción
Este proyecto consiste en una aplicación desarrollada en Python que permite al usuario practicar vocabulario mediante ejercicios interactivos. El sistema se basa en el uso de tarjetas organizadas en mazos y ofrece distintos tipos de ejercicios para reforzar el aprendizaje.

El programa sigue los principios de la Programación Orientada a Objetos (POO), separando responsabilidades en distintas clases como Usuario, Tarjeta, Mazo y tipos de Ejercicio.

---

## Funcionalidades

- Ejercicio de escritura: el usuario introduce manualmente la traducción de una palabra  
- Ejercicio tipo test: el usuario selecciona la opción correcta entre varias posibles respuestas  
- Añadir nuevas tarjetas al mazo  
- Visualización del progreso del usuario:
  - Puntuación total  
  - Número de ejercicios realizados  
  - Número de aciertos  
  - Porcentaje de acierto  
- Persistencia de datos:
  - Guardado automático del usuario y el mazo  
  - Carga de datos al iniciar el programa
